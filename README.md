# 📊 E-Commerce Sales Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** built to analyze the sales performance of a Brazilian e-commerce marketplace. The dashboard transforms raw transactional data into meaningful business insights to understand revenue trends, customer purchasing behavior, product performance, and geographic sales distribution.

The objective of this project is to demonstrate **data analytics, data modeling, and dashboard development skills** using Power BI.

---

## 🎯 Business Objective
Businesses generate large volumes of transactional data but often struggle to convert that data into actionable insights.

This project helps to:

- Analyze **sales performance trends**
- Identify **top-performing product categories**
- Understand **customer purchasing behavior**
- Visualize **regional sales distribution**
- Support **data-driven decision making**

---

## 📂 Dataset
The dataset used in this project is the **Brazilian E-Commerce Public Dataset by Olist**, available on Kaggle.

Dataset Source:  
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset contains information about:

- Customers  
- Orders  
- Order Items  
- Products  
- Payments  
- Reviews

## 📊 Dashboard Pages

### 1️⃣ Executive Overview
This page provides a **high-level summary of business performance**.

**Key Metrics**

- Total Revenue
- Total Orders
- Unique Customers
- Average Order Value (AOV)
- Orders per Customer

**Visualizations**

- Revenue Trend Over Time
- Orders Trend
- Average Order Value Trend
- Interactive Date Filter

---

### 2️⃣ Product & Market Analysis
This page focuses on **product performance and geographic sales distribution**.

**Visualizations**

- Revenue by Product Category
- Top 10 Products by Revenue
- Revenue Distribution by State (Map)
- Category Insights using Interactive Tooltips

**Tooltip Insights**

- Units Sold by Category
- Average Product Price by Category

---

## 📈 Key Business Insights

- Identified **top-performing product categories** driving the highest revenue.
- Analyzed **geographic distribution of sales across Brazilian states**.
- Observed **purchasing patterns and revenue growth trends over time**.
- Highlighted **products contributing most to overall revenue**.
